def checkAdminAccess(user_id, feature_name=""):
    """فحص شامل لصلاحيات الأدمن مع تسجيل تفصيلي"""
    user_id_str = str(user_id).strip()

    if user_id_str == admin_id_str and admin_id_str not in ['', 'YOUR_ADMIN_ID_HERE', '0']:
        return {'access_granted': True, ...}

        return {'access_granted': False, ...}

### 2️⃣ نظام الرسائل الجماعية للأدمن

**الوظيفة:**
- الأدمن يمكنه إرسال رسالة واحدة لجميع المستخدمين
- تسجيل النجاح والفشل لكل مستخدم
- تقرير شامل بعد الإرسال

**الأمر:**
```
/broadcast
```

**كيفية الاستخدام:**
1. الأدمن يرسل `/broadcast`
2. البوت يطلب الرسالة
3. الأدمن يرسل النص
4. البوت يرسل لجميع المستخدمين
5. يُظهر تقرير شامل

**الكود:**
```python
def handle_broadcast_command(chat_id, user_id):
    if not validateAdminCommand(user_id, "/broadcast"):
        send_message(chat_id, "❌ ليس لديك صلاحية")
        return
    
    user_states[user_id] = {'state': 'WAITING_BROADCAST'}
    send_message(chat_id, "📢 أرسل الرسالة التي تريد إرسالها...")

def handle_broadcast_message_input(chat_id, user_id, message_text):
    user_ids = getAllUserIds()
    success_count = 0
    failed_count = 0
    
    for target_user_id in user_ids:
        if send_message(target_user_id, message_text):
            success_count += 1
        else:
            failed_count += 1
    
    # إرسال تقرير للأدمن
    report = f"✅ تم الإرسال لـ {success_count} مستخدم\n❌ فشل {failed_count}"
    send_message(chat_id, report)
```

**الملفات المضافة/المحدثة:**
- `users_manager.py` - دالة `getAllUserIds()`
- `advanced_logger.py` - دالة `logBroadcastOperation()`
- `bot.py` - دوال `handle_broadcast_command()` و `handle_broadcast_message_input()`
