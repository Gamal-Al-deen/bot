# 🚀 دليل الإعداد السريع - بوت SMM المتقدم

## ⚡ خطوات التشغيل في 5 دقائق

### الخطوة 1️⃣: تثبيت المكتبات

```bash
cd bot/bot/bot
pip install -r requirements.txt
```

### الخطوة 2️⃣: إعداد معرف الأدمن

1. افتح Telegram وابحث عن `@userinfobot`
2. أرسل أي رسالة للبوت
3. انسخ المعرف الرقمي (مثال: `123456789`)
4. افتح ملف `config.py`
5. عدّل السطر:
   ```python
   ADMIN_ID = "123456789"  # ضع معرفك هنا
   ```

### الخطوة 3️⃣: التحقق من الإعدادات

افتح `config.py` وتأكد من:

```python
BOT_TOKEN = "8641656014:AAE4n6WqbnusGtWkf5a-RuMXomVRGIN_J5M"  # ✅ موجود
API_KEY = "d4b3feb9bf3d7e4255f4e7d145d43f48"  # ✅ موجود
ADMIN_ID = "YOUR_ADMIN_ID_HERE"  # ⚠️ غيّر هذا!
```

### الخطوة 4️⃣: تشغيل البوت

```bash
python run.py
```

### الخطوة 5️⃣: اختبار البوت

1. افتح البوت على Telegram
2. أرسل `/start`
3. يجب أن تظهر القائمة الرئيسية

---

## ✅ قائمة التحقق بعد التشغيل

### اختبار أوامر المستخدم:

- [ ] `/start` - تظهر القائمة الرئيسية
- [ ] `/balance` - يظهر الرصيد (يجب أن يكون 0)
- [ ] `/services` - تظهر قائمة الخدمات
- [ ] `🔄 تحديث` - يتم تحديث البيانات

### اختبار أوامر الأدمن:

- [ ] `/addbalance YOUR_ID 10` - يجب أن ينجح
- [ ] `/balance` - يجب أن يظهر 10$
- [ ] `/setpercent 50` - يجب أن ينجح
- [ ] `/price` - يجب أن يظهر إعدادات التسعير
- [ ] `/removebalance YOUR_ID 5` - يجب أن ينجح
- [ ] `/balance` - يجب أن يظهر 5$

### اختبار تقديم طلب:

1. أرسل `/neworder`
2. اختر خدمة
3. أرسل رابط صحيح
4. أرسل كمية (مثلاً: 100)
5. راجع ملخص الطلب
6. اضغط `✅ تأكيد`
7. يجب أن يظهر معرف الطلب والرصيد الجديد

## 🔍 استكشاف الأخطاء الشائعة

### ❌ البوت لا يستجيب

```bash
# تحقق من أن البوت يعمل
python run.py

# تحقق من اللوغ
tail -f log.txt
```

### ❌ خطأ في Token

```
تأكد من BOT_TOKEN في config.py
```

### ❌ خطأ في API

```
تأكد من API_KEY في config.py
اختبر API: python test_api_direct.py
```

### ❌ الأدمن لا يعمل

```
تأكد من ADMIN_ID في config.py
يجب أن يكون رقم Telegram ID الخاص بك
```
